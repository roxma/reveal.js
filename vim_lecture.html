简介

  • 80年代末，由 Bram 为他的 Amiga 电脑开发，Vi IMitation
  • 后来 vim 被移植到各个平台，称为 Vi IMproved
  • Vim 被称为 “编辑器之神”，EMACS “神之编辑器”
  • 值得一提的是，2014 年 NeoVim 诞生，发展迅速，也反过来推动了 Vim 社区的发展

编辑命令的格式

[ number ] [ command ] [ text object or motion ]

command

  • y 复制
  • p 粘贴
  • d 删除
  • v/V 选择
  • J 连接
  • = 对齐
  • < > 向左/右边缩进
  • ...

text object

  • iw 词(inside word)
  • ip 段落(inside paragraph)
  • it tag 内容，(inside tag)
  • i( (和) 之间的内容，(inside parenthesis)
  • i{ {和} 之间的内容，(inside braces)
  • aw (和) 之间的内容，包括 ( 和 )，(around parenthesis)
  • ...

常用命令 举几个栗子

  • ci" 修改引号中的内容
  • ciw 重写这个单词，如果用 ce，重写从光标到词末尾的内容
  • c$ 重写本行后面的文本
  • gg=G 格式化整个文件的代码，也可以用 ggVG=
  • vi( / vi) 选中括号中的内容，然后可以复制/粘贴替换/删除(y/p/d/c)...
  • ...

更多技巧

关于复制 & 粘贴

  • :set paste 避免被自动缩进干扰

  • :r!echo "Hello world" 执行外部命令并把结果粘贴到 vim 中

查找

文件内查找

  • /[模式] 模式匹配查找
  • * 查找光标下的单词
  • n 跳转到下一个匹配项
  • N 上一个

模式匹配

  • /hi 匹配 hi
  • /\cHi 强制大小写不敏感。 匹配 HI。
  • /\CHi 强制大小写敏感
  • /\v   Very magic，类似正则表达式，除 '0'-'9', 'a'-'z', 'A'-'Z' 和 '_' 以外的 ASCII 字符都具备特别含义。

\v \m \M \V

\v \m (magic) \M ('nomagic') \V matches
$ $ $ \$ matches end-of-line
. . \. \. matches any character
\. \. . . literal dot
* * \* \* any number of the previous atom
| \| \| \| separating alternatives

:help \v 查看更多信息

有用的配置

  • 默认 very magic，区分大小写nnoremap / /\c\v
  • 实时高亮：set incsearch
  • 高亮匹配项：set hlsearch
  • 双击 ESC 取消当前的高亮：nnoremap <ESC><ESC> :let @/ = ""<CR>

FZF 插件

  • :FZF [name] 模糊查找文件
  • :FZFBuffers [name] 查找并选择打开的文件
  • :FZFAg [text] 在文件系统中查找文本
  • :FZFLines [text]模糊查找打开的文件的内容
  • 按键映射，如 nnoremap <C-f><C-f> :FZF<CR>

顺便说一下替换...

  • 批量替换法

    1. 用 V 选择需要操作的区域
    2. 输入 :s/hello/Hello/g 将区域内的 hello 替换成 Hello

  • 逐个确认法

    1. 用 /hello 查找 hello
    2. 键入 ceHello 将 hello 替换
    3. n 定位到下一个 hello
    4. 按下 . 重复替换命令
    5. 重复步骤三，步骤四

多窗口管理

  • 水平/垂直分割
  • Tab

水平/垂直分割

  • :split [文件名] 水平分割出上下两个窗口
  • :vsplit [文件名] 垂直分割出左右连个窗口
  • Ctrl+w h/j/k/l 移动光标到 左/下/上/右 边的窗口
  • Ctrl+w +/-/_/| 调整窗口大小
  • :q 关闭窗口

建议技巧

  • Ctrl+h/j/k/l 快速切换窗口 nnoremap <C-h> <C-w>h
    nnoremap <C-j> <C-w>j
    nnoremap <C-k> <C-w>k
    nnoremap <C-l> <C-w>l
  • winresizer 插件，快速调整窗口大小

Tab

  • :tabnew [文件名]
  • :tabnext 下一个 Tab
  • :tabprev 上一个 Tab
  • :q 关闭 Tab

建议技巧

  • H/L 左右快速切换 Tab nnoremap <S-l> :tabn<CR>
    nnoremap <S-h> :tabN<CR>
